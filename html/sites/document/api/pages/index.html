BrawlStars Character API - ドキュメント

エンドポイント：キャラクター情報取得

GET /api/character

認証

このエンドポイントには トークンが必須 です。
トークンは 公式サイト にて取得してください。

パラメータ（Query）

パラメータ 必須 説明 例
token 認証用JWT。サイトで生成してください。 eyJhbGciOiJIUzI1Ni...
lang キャラクターの翻訳言語を指定します。日本語はja、英語はen。 en (デフォルト: ja)
rarity 取得したいキャラクターのレア度でフィルタリングします。
character.json に記述されている正確なレア度名を使用してください。		 ウルトラ, レジェンド
role 取得したいキャラクターのロール（タイプ）でフィルタリングします。
character.json に記述されている正確なロール名を使用してください。		 アタッカー, タンク
count 取得したいキャラクターの数を指定します。 3 (デフォルト: 1)
blockedIds 結果に含めたくないキャラクターのIDをカンマ区切りで指定します。 char_001,char_005

正確なキャラクターIDの確認方法:

各キャラクターの正確なIDは、以下のファイルをダウンロードしてご確認ください。

./html/sites/document/api/pages/Character.txt をダウンロードしてください。

リクエスト例

ランダムなキャラクターを1体、日本語で取得:

GET /api/character?token=あなたのトークンを記載してください...
Host: api-brawlstars-character.vercel.app

またはブラウザで：

https://api-brawlstars-character.vercel.app/api/character?token=あなたのトークンを記載してください

レア度が「レジェンド」のキャラクターを1体取得 (英語):

GET /api/character?token=YOUR_TOKEN&lang=en&rarity=レジェンド
Host: api-brawlstars-character.vercel.app

またはブラウザで：

https://api-brawlstars-character.vercel.app/api/character?token=YOUR_TOKEN&lang=en&rarity=レジェンド

ロールが「アサシン」のキャラクターを2体取得し、特定のID (例: char_057 と char_078) をブロック:

GET /api/character?token=YOUR_TOKEN&role=アサシン&count=2&blockedIds=char_057,char_078
Host: api-brawlstars-character.vercel.app

またはブラウザで：

https://api-brawlstars-character.vercel.app/api/character?token=YOUR_TOKEN&role=アサシン&count=2&blockedIds=char_057,char_078

レスポンス形式（JSON）

単一のキャラクターが返される場合 (count が指定されないか 1 の場合):

{
  "name": "キャラクター名",
  "rarity": "レア度",
  "role": "タイプ"
}

複数のキャラクターが返される場合 (count が 2 以上の場合):

[
  {
    "name": "キャラクター名1",
    "rarity": "レア度1",
    "role": "タイプ1"
  },
  {
    "name": "キャラクター名2",
    "rarity": "レア度2",
    "role": "タイプ2"
  }
]

その他の注意点:

  • rarity または role でフィルタリングする場合、character.json に記述されている正確な日本語名を使用してください。
  • blockedIds はカンマ区切りの文字列で指定します。
  • URLの長さには制限があるため、blockedIds が非常に長くなる場合はご注意ください。

エラーレスポンス例

ステータス 内容
400 {"error":"Token is required"}
401 {"error":"Invalid or expired token"}
404 {"error":"No characters found with the specified criteria after filtering and blocking."}
指定された条件（レア度、タイプ、ブロックIDなど）に合致するキャラクターがいない場合。
