BrawlStars Character API - ドキュメント

エンドポイント：ランダムキャラクター取得

GET /api/character/random

認証

このエンドポイントには トークンが必須 です。
トークンは 公式サイト にて取得してください。

パラメータ

このAPIはトークン以外のパラメータは不要です。

パラメータ 必須 説明 例
token✅認証用JWTeyJhbGciOiJIUzI1Ni...

リクエスト例

GET /api/character/random?token=YOUR_TOKEN
Host: api-brawlstars-character.vercel.app
https://api-brawlstars-character.vercel.app/api/character/random?token=YOUR_TOKEN

レスポンス形式

ランダムに取得した単一キャラクター情報：

{
  "name": "キャラクター名",
  "rarity": "レア度",
  "role": "タイプ"
}
注意: トークンは必ず有効なものを使用してください。
