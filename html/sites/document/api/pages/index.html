BrawlStars Character API - ドキュメント

エンドポイント：キャラクター情報取得

GET /api/character

認証

このエンドポイントには トークンが必須 です。
トークンは 公式サイト にて取得してください。

パラメータ（Query）

パラメータ 必須 説明 例
token✅認証用JWTeyJhbGciOiJIUzI1Ni...
lang❌翻訳言語指定en
rarity❌キャラのレア度で絞り込みレジェンド
role❌キャラのロールで絞り込みアサシン
count❌取得するキャラ数3
blockedIds❌除外したいキャラIDchar_001,char_005

正確なキャラクターIDの確認方法:

以下のファイルをダウンロードして確認できます。

Character.txt をダウンロード

リクエスト例

GET /api/character?token=YOUR_TOKEN
Host: api-brawlstars-character.vercel.app
https://api-brawlstars-character.vercel.app/api/character?token=YOUR_TOKEN

レスポンス形式

単一キャラクター：

{
  "name": "キャラクター名",
  "rarity": "レア度",
  "role": "タイプ"
}

複数キャラクター：

[
  {
    "name": "キャラ1",
    "rarity": "レア1",
    "role": "タイプ1"
  },
  {
    "name": "キャラ2",
    "rarity": "レア2",
    "role": "タイプ2"
  }
]
注意: rarity や role の指定には JSON ファイルにある正式名称を使ってください。
